\CWHeader{Архиватор (Huffman + BWT + MTF + RLE)}

\CWProblem{

Необходимо реализовать четыре известных метода пребрзования данных для сжатия одного файла.\\

Формат запуска должен быть аналогичен формату запуска программы gzip, должны быть поддержаны следующие ключи: -c, -d, -k, -l, -r, -t, -1, -9. Должно поддерживаться указание символа дефиса в качестве стандартного ввода. \\

Ключи:
\begin{itemize}
    \item {\bfseries -c} - произести вывод результата в стандартный вывод
    \item {\bfseries -k} - не удалять входной файл после сжатия
    \item {\bfseries -l} - вывести информацию о сжатом файле
    \item {\bfseries -d} - разархивирование файла
    \item {\bfseries -r} - рекурсиный проход по директориям с архивироанием лежащих файлов
    \item {\bfseries -t} - проверка целостности файла
    \item {\bfseries -1} - быстрое сжатие
    \item {\bfseries -9} - глубокое сжатие
    
\end{itemize} \\

Помимо ключей при запуске следует указать пути к архивируемым файлам. Если пути не указаны, программа должна считать данные из стандартного поток ввода.

}
